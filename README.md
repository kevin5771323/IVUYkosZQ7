# 项目简介

## 核心功能
- 提供完整的煤炭传送带异物检测系统源码
- 包含2300张标注数据集，涵盖['foreign','block','crack','hole']四类异物
- 提供训练好的模型及训练结果
- 支持图片、视频及摄像头进行实时检测
- 提供UI界面，可实时显示目标位置、目标总数、置信度等信息
- 支持批量检测功能，可查看所有检测结果
- 支持检测结果保存
- 允许自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 提供项目运行教程和环境配置说明

## 适用场景
- 煤炭传送带异物实时监测
- 工业安全检测系统
- 自动化生产线质量控制
- 物料传输安全监控

## 技术特点
- 基于YOLOv8深度学习框架
- 使用Python 3.9开发环境
- 集成OpenCV、PyQt5等技术
- 模型训练完成，可直接部署使用
- 提供完整的开发环境和UI界面
